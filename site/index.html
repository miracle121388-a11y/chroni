常见问题

开始前可能关心的事

一定要配置大模型 API Key 吗？

不需要。结构清楚的文本和日期可以由本地规则完成基础提取；需要理解长文档、复杂描述或图片语义时，再连接 DeepSeek 等兼容服务。

安装包里需要另外安装 Node.js 吗？

不需要。公开安装包已经包含运行环境，普通用户下载对应系统版本后即可启动。

为什么下载仍然来自 GitHub？

官网负责识别系统并把你直接带到正确文件，GitHub 只在后台提供可追溯的安装包、校验文件和构建来源。
