Deadline Agent

它先替你看一遍。

剩余工时够不够、哪个任务风险最高、今天还能安排多少，Agent 会在计划进入日程前完成一次复查。

  • 综合截止时间、依赖和每日容量
  • 保留规划来源与结构化执行记录
  • 模型不可用时仍有本地规则回退

Before you install

开始前可能关心的事。

一定要配置大模型 API Key 吗？

不需要。结构清楚的文本和日期可由本地规则完成基础提取；需要理解长文档、复杂描述或图片语义时，再连接 DeepSeek 等兼容服务。

需要另外安装 Node.js 或 pnpm 吗？

不需要。公开安装包已经包含运行环境，普通用户下载对应系统版本后即可启动。

为什么下载文件仍来自 GitHub？

GitHub Release 提供公开可追溯的安装包、校验文件与构建来源；这个网站负责识别系统并把你直接带到正确文件。

系统拦截未签名安装包怎么办？

当前公开版本可能触发 SmartScreen 或 Gatekeeper。只从本页下载，并按照安装 FAQ 核对文件名和 SHA-256；不要关闭系统全局安全机制。
