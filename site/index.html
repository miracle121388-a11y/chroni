Before you install

开始前可能关心的事。

一定要配置自己的大模型 API Key 吗？

不需要。内测期间可使用 Chroni 内测访问码连接托管模型服务；也可以切换到自定义 API，使用自己的 DeepSeek 等兼容服务。

文件会上传到 Chroni 服务器吗？

文件解析、PDF 文本提取和图片 OCR 在桌面端完成。开启智能模型时，当前任务所需的提取文本会发送给所选模型服务，原始文件不会进入 Chroni 网关。

Windows 和 macOS 的功能一致吗？

两端共享日程、Agent、每日计划和文件提取能力，并分别适配窗口、桌宠拖动、托盘与系统通知交互。

新版本怎样更新？

本页会通过 GitHub Releases 自动读取最新正式版本。发布完成后，下载按钮会自动指向新的安装包。
