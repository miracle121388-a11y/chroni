# Chroni v0.1.4 — 课程项目最终交付版

Chroni 是一个 local-first 混合式 Deadline Agent：把通知、文档、表格、日历和截图中的截止事项，转成经过证据校验的任务、可编辑 TaskPlan、每日时间块与桌面提醒。LLM 负责提出结构化候选，本地系统负责事实校验、风险与容量计算、状态变更、工具调用、Trace、Memory 和失败回退。

## 三分钟演示

1. 启动 Chroni，需要展示模型能力时在“偏好 -> 高级 -> 大模型 API”保存并测试 DeepSeek Key。
2. 输入“今天晚上八点提交课程项目”（20:00 后演示请改为“明天”）。
3. Chroni 应直接生成标题“课程项目”和 20:00 截止时间，不追问已经明确的字段。
4. 检查并启用 TaskPlan，在 Agent 工作台点击“帮我安排今天”。
5. 在“为什么这样安排”中查看 `Observe -> Plan -> Act -> Verify` Trace 和本次规划来源，再到每日时间轴执行或调整工作块。

- 使用 Source Serif 4 / Noto Serif SC 与 Source Sans 3 / Noto Sans SC 建立中英文和数字排版层级。
- README 新增课程要求映射、混合 Agent 职责边界、完整架构图、演示路径、质量证据、隐私范围和已知边界。
- 修正安装版与源码 `.env` 的 API 配置说明，并披露自动模型调用与费用触发点。
- 完整保留 XIAOTONG Desktop Pet 的原作 About 信息、捐赠二维码与许可证入口；发行包外置附带第一方与第三方许可证。

## API、隐私与许可证提示

- 安装版请从控制中心配置 API；`.env` 仅供源码 `dev/start` 使用。
- 开启 LLM 后，解析 / OCR 文本和规划所需的结构化上下文会发送到用户配置的服务；二进制原文件不会直接上传。
