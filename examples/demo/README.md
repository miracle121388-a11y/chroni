# Chroni 演示材料

这些文件只包含虚构内容，用于第一次体验、录屏和问题复现。它们不是应用内 Demo Mode，不会自动写入用户数据。

## 使用方式

1. 启动 Chroni。
2. 把一个 `.txt` 文件拖到桌宠，或在控制中心“日程”中选择“预览抽取”。
3. 核对识别结果后再填入日程。
4. 打开规划详情并启用 TaskPlan。
5. 在 Agent 页面点击“帮我安排今天”。

## 三份材料

- `01-course-assignment.txt`：明确标题、截止时间、提交物和提交方式，适合无 Key 本地演示。
- `02-innovation-pitch.txt`：多项交付物、展示和协作要求，更适合测试模型增强。
- `03-internship-application.txt`：多个截止节点与缺失信息，适合展示待确认和风险处理。

示例使用“明天”“本周日”“下周五”等相对时间。录屏前请核对电脑日期和最终解析日期，避免跨午夜或周界造成歧义。

演示结束后，可在日程中删除对应 DDL 和每日任务。当前版本没有隔离式 Demo 数据空间。
